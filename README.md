# HotProspects
Conference Networking iOS application done as part of Hacking with SwiftUI coursework: https://www.hackingwithswift.com/100/swiftui

# Description
This SwiftUI application allows sharing contact information (name and e-mail) via QR code. The app creates a code for the user and stores a list of all contacts scanned.
The app applies: Custom Environment Objects, TabView, Image Interpolation, Context Menus, Local Notifications, Swift Package Dependency, filter() and map().
